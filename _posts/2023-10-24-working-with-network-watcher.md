Last week I held [Working with Azure Network Watcher](https://www.meetup.com/infrastructure-as-code-user-group-oslo/events/294715449) master class under my [Infrastructure as Code user group](https://www.meetup.com/Infrastructure-As-Code-User-Group-Oslo).

This time we looked at different aspects aspects of working with Azure Network Watcher and you will learn:

- How to enable Network Watcher
- How to diagnose outbound connection problems
- How to diagnose a routing problems
- How to monitor network performance
- How to log network traffic
- How to capture and analyze packets

and much more...
